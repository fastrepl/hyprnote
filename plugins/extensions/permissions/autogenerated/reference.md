## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-load-extension`
- `allow-call-function`
- `allow-execute-code`
- `allow-list-extensions`
- `allow-get-extensions-dir`
- `allow-get-extension`
