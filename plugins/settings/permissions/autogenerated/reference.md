## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-settings-base`
- `allow-content-base`
- `allow-change-content-base`
- `allow-obsidian-vaults`
- `allow-path`
- `allow-load`
- `allow-save`
