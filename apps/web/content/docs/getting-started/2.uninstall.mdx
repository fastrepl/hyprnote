## macOS

To uninstall Hyprnote:

1. Quit Hyprnote completely by clicking on the tray icon and the menubar icon, then click "Quit completely"

<Image src="/api/images/docs/getting-started/tray-icon-menu.jpg" />

2. Move Hyprnote to the trash bin
3. If you want to remove the data files that were generated by Hyprnote, run the following command:

```bash
rm -rf ~/Library/Application\ Support/hyprnote
```
