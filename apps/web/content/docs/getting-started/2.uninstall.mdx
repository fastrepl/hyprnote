## macOS

To uninstall Char:

1. Quit Char completely by clicking on the tray icon and the menubar icon, then click "Quit completely"

<Image src="/api/images/docs/getting-started/tray-icon-menu.jpg" />

2. Move Char to the trash bin
3. If you want to remove the data files that were generated by Char, run the following command:

```bash
rm -rf ~/Library/Application\ Support/hyprnote
```
