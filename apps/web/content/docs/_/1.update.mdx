## Automatic Updates

Hyprnote checks for updates automatically. When a new version is available, you'll see a notification in the app.

Click the notification or go to **Settings** to download and install the update.

## Manual Update

You can also manually check for updates:

1. Open Hyprnote
2. Go to **Settings**
3. Click **Check for Updates**

If an update is available, follow the prompts to install it.

## Release Channels

Hyprnote has two release channels:

- **Nightly**: Latest features and fixes, updated frequently
- **Stable**: Thoroughly tested releases (coming soon)

You can switch between channels in **Settings**.
