## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-list-devices`
- `allow-list-input-devices`
- `allow-list-output-devices`
- `allow-get-default-input-device`
- `allow-get-default-output-device`
- `allow-set-default-input-device`
- `allow-set-default-output-device`
- `allow-is-headphone`
- `allow-load-state`
- `allow-save-state`
- `allow-get-input-priorities`
- `allow-get-speaker-priorities`
- `allow-get-headphone-priorities`
- `allow-save-input-priorities`
- `allow-save-speaker-priorities`
- `allow-save-headphone-priorities`
- `allow-move-device-to-top`
- `allow-get-device-category`
- `allow-set-device-category`
- `allow-get-current-mode`
- `allow-set-current-mode`
- `allow-is-custom-mode`
- `allow-set-custom-mode`
- `allow-get-known-devices`
- `allow-remember-device`
- `allow-forget-device`
- `allow-is-device-hidden`
- `allow-hide-device`
- `allow-unhide-device`
