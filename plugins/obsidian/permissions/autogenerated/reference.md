## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-api-key`
- `allow-get-base-url`
- `allow-get-vault-name`
- `allow-set-api-key`
- `allow-set-base-url`
- `allow-set-vault-name`
- `allow-get-enabled`
- `allow-set-enabled`
- `allow-get-deep-link-url`
- `allow-get-base-folder`
- `allow-set-base-folder`
