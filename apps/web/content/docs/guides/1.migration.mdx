## Migrating from Other Tools

Hyprnote supports importing notes from various sources. This guide covers how to migrate your existing data.

## Supported Formats

You can import notes in the following formats:

- **Markdown** (`.md`, `.mdx`)
- **Plain Text** (`.txt`)
- **JSON** (structured export from other apps)

## Import Process

1. Open Hyprnote and navigate to Settings
2. Select "Import" from the data section
3. Choose your files or folder to import
4. Review the import preview
5. Confirm to complete the migration

## From Specific Tools

### From Otter.ai

Export your transcripts from Otter.ai as text files, then import them into Hyprnote using the import feature.

### From Fireflies.ai

Download your meeting notes from Fireflies.ai and import the text or markdown files directly.

### From Apple Notes

Use the export functionality in Apple Notes to save your notes as text, then import into Hyprnote.
