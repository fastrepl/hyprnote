Hyprnote supports 45+ languages for transcription and AI features. Language support varies by provider.

## Supported Languages

See the [Features FAQ](/docs/faq/features) for the full list of supported languages.

## Provider-Specific Support

Each transcription provider has different language coverage:

| Provider | Languages | Notes |
|----------|-----------|-------|
| Whisper (local) | 50+ | Best multilingual support |
| Deepgram | 30+ | [Full list](https://developers.deepgram.com/docs/models-languages-overview) |
| AssemblyAI | 20+ | Best for English variants |

## Configuration

Set your main language and additional spoken languages in Settings > Language & Vocabulary. The main language is used for AI responses; spoken languages improve transcription accuracy in multilingual meetings.
