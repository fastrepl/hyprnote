Admin server is a server-side application that Hyprnote desktop app can connect to. It provides:

- Integrations (OAuth, Webhook)
- Sharing & Syncing data
- User Management
- Etc

We're still working on it. Feel free to reach out to us if this is something you want to use within an organization.
