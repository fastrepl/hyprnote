## Navigation

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd> + <kbd>1</kbd>, <kbd>2</kbd>, <kbd>3</kbd>, ... | Switch to specific outer tab |
| <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd> / <kbd>→</kbd> | Navigate to next/previous outer tab |
| <kbd>⌃</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd> / <kbd>→</kbd> | Navigate to next/previous inner tab |

## Sidebar & Panels

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd> + <kbd>\\</kbd> | Toggle left sidebar |
| <kbd>⌘</kbd> + <kbd>J</kbd> | Toggle AI assistant |

## Notes & Tabs

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd> + <kbd>N</kbd> | Create new note |
| <kbd>⌘</kbd> + <kbd>T</kbd> | Open new tab |
| <kbd>⌘</kbd> + <kbd>W</kbd> | Close current tab |

## Quick Access

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>C</kbd> | Open calendar |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>N</kbd> | Open contacts |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>D</kbd> | Open daily notes |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>F</kbd> | Open advanced search |

## Application

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd> + <kbd>Q</kbd> | Quit Hyprnote |
