## Default Permission

Default permissions for the plugin

- `allow-is-server-running`
- `allow-is-model-loaded`
- `allow-is-model-downloaded`
- `allow-download-model`
- `allow-load-model`
- `allow-unload-model`
- `allow-start-server`
- `allow-stop-server`
