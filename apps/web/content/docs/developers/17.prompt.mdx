# Developemt

- In `VSCode-like` editor, [Better Jinja](https://marketplace.visualstudio.com/items?itemName=samuelcolvin.jinjahtml) extension is recommended.
- We use `.md.jinja` extension for `Jinja Markdown` highlighting. If it doesn't work, you can change the [language manually](https://code.visualstudio.com/docs/languages/overview#_change-the-language-for-the-selected-file).
