Hyprnote includes a CLI for automation and scripting. Access it via the app bundle:

```bash
# macOS
/Applications/Hyprnote.app/Contents/MacOS/hyprnote-cli --help
```

## Available Commands

Run `--help` on any command for detailed usage. The CLI supports deeplinks, session management, and integration with shell scripts.

See [Hooks](/docs/developers/hooks) for automating workflows based on Hyprnote events.
