## Default Permission

Default permissions for the plugin

- `allow-start-oauth-server`
- `allow-stop-oauth-server`
- `allow-init-vault`
- `allow-reset-vault`
- `allow-get-from-store`
- `allow-get-from-vault`
- `allow-set-in-store`
- `allow-set-in-vault`
