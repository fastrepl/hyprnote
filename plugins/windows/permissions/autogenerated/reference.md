## Default Permission

Default permissions for the plugin

- `allow-window-show`
- `allow-window-destroy`
- `allow-window-position`
- `allow-window-get-floating`
- `allow-window-set-floating`
- `allow-window-navigate`
- `allow-window-emit-navigate`
