## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-git-hash`
- `allow-get-fingerprint`
- `allow-opinionated-md-to-html`
- `allow-audio-open`
- `allow-audio-exist`
- `allow-delete-session-folder`
- `allow-parse-meeting-link`
