import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Buy Pro License**

   You'll receive a confirmation email with your license key.
   
   ![Hyprnote Activation Step 1](/api/images/docs/pro/activation-1.png)
   
   The license key will look like this:

   ```bash
   key/******************************************
   ```

2. **Go to settings**

   Open Hyprnote and navigate to Settings. Paste the license key into the text input.
   
   ![Hyprnote Activation Step 2](/api/images/docs/pro/activation-2.png)

</Steps>
