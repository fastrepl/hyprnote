## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-open-calendar`
- `allow-list-calendars`
- `allow-list-events`
- `allow-create-event`
- `allow-advance-fixture`
- `allow-reset-fixture`
- `allow-get-fixture-info`
